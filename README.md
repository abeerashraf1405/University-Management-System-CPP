# University Management System (C++)

A comprehensive, console-based management system engineered using custom Data Structures and Algorithms.

## 🚀 Features
* **User Authentication**: Secure login/registration using a **Hash Table** with chaining.
* **Campus Mapping**: Navigation and building connectivity using a **Graph** (BFS/DFS traversals).
* **Room Scheduling**: Efficient room management and reservation via **AVL Trees** for $O(\log n)$ searching.
* **Complaint System**: FIFO-based issue tracking using a custom **Queue**.
* **Messaging**: A stack-based messaging module for user communication.

## 🛠️ Tech Stack
* **Language**: C++
* **Data Structures**: Hash Table, AVL Tree, Graph (Adjacency List), Queue, Stack, Linked List.
* **Tools**: Standard Template Library (minimal), Custom Logic.

## 💻 How to Run
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/University-Management-System-CPP.git`
2. Compile the code:
   `g++ UMS.cpp -o UMS`
3. Run the executable:
   `./UMS`
